# Bank Application — System Analysis & Design

A complete system analysis and design case study for a banking application, covering the full software design lifecycle using UML modeling.

## Overview
This project focuses on the **design phase** of software engineering: analyzing requirements and modeling a banking system using standard UML diagrams before implementation.
